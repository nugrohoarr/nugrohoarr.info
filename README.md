
## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/nugrohoarr/portofolio-v1.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portofolio-v1
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Run the development server:
    ```bash
    npm run dev
    ```
