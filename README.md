
## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/r2hu1/DevPortfolio.git
    ```
2. Navigate to the project directory:
    ```bash
    cd DevPortfolio
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Run the development server:
    ```bash
    npm run dev
    ```
